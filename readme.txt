This program takes a single file as input named schedule.txt in the same directory. Schedule Items have an option for categories but they are not currently used. The program will load up your list of items and then schedule them randomly (and completely randomly at the moment).

Input file format requirements below. Line items are separated by comma and categories are separated by equal sign. Categories are not yet used.

CATEGORY
Line Item, <Duration in minutes>
=

For example:
LEISURE
Play video games, 60
Read a book, 30
Watch TV, 30
=

WORK
Write report, 15
Host meeting, 30
Debug programs, 60
=
